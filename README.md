# Hybrid AI + ML Journey

Профессиональный путь к Middle Hybrid AI + ML Engineer. Проект включает в себя инженерные практики, математический фундамент и современные подходы AI.

## Инфраструктура
- **Python**: 3.10
- **Environment**: Conda (hybrid_ai)
- **Automation**: Makefile

## Как запустить
1. Склонируйте репозиторий.
2. Установите зависимости: `make install`
3. Проверьте код: `make lint`
